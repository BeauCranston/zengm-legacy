Edit Team Info

You can manually edit the teams below or you can upload a teams file to specify all of the team info at once.

Upload Teams File

The JSON file format is described in the manual. As an example you can download a teams file containing the current default teams.

On average, the team with the highest starting prestige will have the best team to start with. The team with the lowest starting prestige will be the worst.

It takes 3 seasons for a team to move. So any location changes (latitude and longitude) will only take effect for newly created recruiting classes (the game stores 3 years worth).

Warning: selecting a valid team file will instantly apply the new team info to your league.
